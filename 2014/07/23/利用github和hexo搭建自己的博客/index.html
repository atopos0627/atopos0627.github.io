如何在一个全新的电脑系统中添加自己的博客日志,利用hexo和github.

在一个新的电脑系统上,按照如下步骤执行即可:

更新github的SSH密钥

检查现有的SSH密钥 cd ~/. ssh 检查本机的ssh密钥，不存在该文件夹就说明未配置过该密钥
生成新的SSH密钥 ssh-keygen -t rsa -C “邮件地址@youremail.com”
将id_rsa.pub复制到Github中即可。
设置用户信息 git config —global user.name “cnfeat”//用户名
git config —global user.email “cnfeat@gmail.com”//填写自己的邮箱

安装 git 和 node js

安装部署 hexo

打开git，执行 npm install -g hexo
在我的电脑中创建hexo文件夹，右键该文件夹，打开Git Bash，执行 hexo init 和 npm install。Hexo随后会自动在目标文件夹建立网站所需要的所有文件。
hexo g
hexo s ，即可 输入 localhost:4000 查看

独立域名与博客关联

godaddy注册购买域名
DNSPod 修改DNS
hexo 更新源代码，并添加CNAME文件

写博客

将hexo文件夹解压，执行git bash
hexo new *.md
修改tags
hexo g
hexo s

更新到github

hexo d

主题选择

一直喜欢简洁，清楚明白的表达，博客也是这样，所以选择了strict的黑白主题。
需要添加点评论，统计或者自己的介绍啥的，网上找了找，觉得这个哥们在strict上修改的主题不错，就借用了，
并保留了该哥们的链接。链接请点击：博客从wordpress迁到hexo 托管在github
不过这个主题有些地方需要稍微修改下：在layout下post.jade中，将 去掉config.disqus_shortname
还有一个地方修改我忘记了，反正我修改了下后，感觉还不错。对了，要在主题的根目录下的config文件中注释掉disqus_shortname。
