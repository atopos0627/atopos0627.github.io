最新的Android源码下载，编译等过程中的完整细节，留下的都是干货。

编译前环境配置

  • 操作系统 Ubuntu 12.04
  • 硬盘空间 大于9G

  • Python 2.7, GNU Make 3.81 or 3.82, Git 1.7 or later, and the Oracle (Sun) Java Development Kit version 6.

    sudo apt-get install git-core gnupg flex bison gperf build-essential \
zip curl libc6-dev libncurses5-dev:i386 x11proto-core-dev \
libx11-dev:i386 libreadline6-dev:i386 libgl1-mesa-glx:i386 \
libgl1-mesa-dev g++-multilib mingw32 openjdk-6-jdk tofrodos \
python-markdown libxml2-utils xsltproc zlib1g-dev:i386

  • 为重定向错误的库

    sudo ln -s /usr/lib/i386-linux-gnu/mesa/libGL.so.1 /usr/lib/i386-linux-gnu/libGL.so

  • 采用缓存（可选，但很有利于编译）

    编辑 ~/.bashrc or ~/.profile file 
添加 export USE_CCACHE=1
  • 改变输出路径到其他的文件系统(针对Jelly Bean (4.1) 和更新版本有效)
    export OUT_DIR_COMMON_BASE=

准备REPO

  • 下载REPO

    mkdir ~/bin
export PATH=~/bin:$PATH

curl $ http://commondatastorage.googleapis.com/git-repo-downloads/repo > ~/bin/repo
curl https://dl-ssl.google.com/dl/googlesource/git-repo/repo > ~/bin/repo
chmod a+x ~/bin/repo

  • 初始化 

    修改第五行 https 为 http
./repo init -u https://android.googlesource.com/platform/manifest/+/refs/heads/android-5.0.1_r1
./repo init -u https://android.googlesource.com/platform/manifest -b android-4.4_r1.2.0.1
