备考申论等考试过程中的学习笔记.

《钟君讲申论》1-22讲总结

  1. 口号：成功励志，幸福成公
  2. 道 述 势
    • 道 - 概述，思维
    • 述 - 解题方法，将概述中的思想转换为方法
    • 势 - 热点问题，逻辑，顺着线索
  3. 申论的境界：五有达人
    • 有脑：会思考 - 申论万能思维体系
    • 有种：展特色 - 和而不同，活学活用
    • 有料：有内涵 - 材料+积累 => 社会现实
    • 有型：擅表达 - 逻辑+语言 （红利+红线） 形象的表达：舌尖上的。。。
    • 有门：懂技巧 概括／分析／对策／写作题
  4. 申论的黑洞
    • 有材料，没套路
    • 有套路，没思想
    • 有思想，难表达 （人才：队伍，干部，编制）

  5. 形象的比喻

  6. 申论是写出来的，面试是写出来的

  7. 静思，悟道。 什么是浮躁：不知以为知，少知不求知
  8. 思维很重要！中国文化最根本的是修心。 富贵：身安为富，心安为贵。
  9. 我们到底要学什么？鸡蛋要从哪里打破？先放石头？先放沙子？
  10. 知识不是力量，智慧才是力量！
  11. 教育就是教人去思考。洞察力比记忆力更重要，思考力比记忆力更重要。
  12. 目无全牛 - 找到其内在的逻辑
  13. 申论考的是什么？ 透过现象看本质，思维能力，情商。
  14. 做一个什么样的学生？
    • 奴：被动的存在
    • 徒：主动，但什么也不知道
    • 工：主动，懂一点皮毛
    • 匠：精细，精致。别人让做，照图纸，没有自己的创新
    • 师：把握规律
    • 家：无人能及
    • 圣：人生 ，全部打通
  15. 申论的道：述材料。 出题者的思路 => 材料 => 答案
  16. 申论考试命题和答题的逻辑：命题专家的抽象 => 演绎成材料的具体 => 考生答案的抽象
  17. 申论答题的问题1: 碎片化的答案。
  18. 申论答题的问题2: 银丝提取不能破坏红木。
  19. 申论辩证法的三个关键词：多分法，结合，具体问题具体分析
  20. 申论的万能思维体系
    • 实务维度：
      • 主体
        • 构成
          • 政府：公共部门的第一部门，其他还有事业单位和非营利组织。政府的价值追求是：公平公正。
          • 企业：市场领域。价值追求：效率。
          • 群众：社会。价值追求：权利。
        • 方面（发挥作用）
          • 利益：即需要
            • 经济利益：钱，物质
            • 政治利益：权利，权力
            • 文化利益：文化
              • 核心层：价值观，精神，道德（2012联考：动物伦理；2014国考：社会心态，幸福，慢生活；2014年4月联考：文化输出和文化拿来）
              • 载体：物质和非物质（2013国考：文化遗产的保护）
              • 制度：体制，机制。
                • 风俗习惯
                • 法律，制度
                  • 文化事业：公共文化服务
                  • 文化产业：市场性产品
                  • 文化交流：2012.9 联考
            • 社会利益：民生
            • 生态利益
          • 思想
            • 理念
            • 意识，责任，大局，环保
            • 常识，知识
          • 行为素质
            • 常识，知识
            • 道德，身体，业务素质
            • 心理素质
      • 制度
        • 规则：法律法规，静态，单一，规定
        • 机制：系统，动态的制度，实用化
        • 体制：雷区，政治体制不要碰。（宪政，普世价值，价值事实，价值观）管理体制的三大问题
          1. 多头管理。如食品安全中，工商，药监。。。
          2. 权责不对等。城管很冤枉：相对集中的行政处罚权， 近400项职责
          3. 职责不清。新一届政府提出”权力清单“。
      • 技术
        • 硬技术
          • 技术手段（物联网）
          • 人：人才，队伍，干部，专家，编制
          • 财：资金，经费，支出，投入，预算。预算分三种：
            • 财政投入
            • 银行贷款，发债
            • 利用民间资本，社会资本（BT／BOT项目）
          • 物：设施，设备，机构，组织
        • 软技术：管理
          • 规划，决策
          • 施行，执行
          • 监督，检查，评估（标准）
          • 总结，反思
      • 环境
        • 人文环境，社会氛围
        • 自然环境
    • 时间维度
      • 过去，现在，未来
      • 事前，事中，事后
    • 空间维度
      • 物理空间
        • 民族
        • 世界
      • 思维空间
        • 内因，外因。如三农问题：输血式：贷款；造血式：农民主动性
        • 主观，客观
    • 价值维度
      • 利
      • 弊
