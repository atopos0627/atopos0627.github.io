备考申论等考试过程中的学习笔记.

《钟君讲申论》务虚部分：申论的整体把握

  1. 口号：成功励志，幸福成公
  2. 道 述 势
    • 道 - 概述，思维
    • 述 - 解题方法，将概述中的思想转换为方法
    • 势 - 热点问题，逻辑，顺着线索
  3. 申论的境界：五有达人
    • 有脑：会思考 - 申论万能思维体系
    • 有种：展特色 - 和而不同，活学活用
    • 有料：有内涵 - 材料+积累 => 社会现实
    • 有型：擅表达 - 逻辑+语言 （红利+红线） 形象的表达：舌尖上的。。。
    • 有门：懂技巧 概括／分析／对策／写作题
  4. 申论的黑洞
    • 有材料，没套路
    • 有套路，没思想
    • 有思想，难表达 （人才：队伍，干部，编制）

  5. 形象的比喻

  6. 申论是写出来的，面试是写出来的

  7. 静思，悟道。 什么是浮躁：不知以为知，少知不求知
  8. 思维很重要！中国文化最根本的是修心。 富贵：身安为富，心安为贵。
  9. 我们到底要学什么？鸡蛋要从哪里打破？先放石头？先放沙子？
  10. 知识不是力量，智慧才是力量！
  11. 教育就是教人去思考。洞察力比记忆力更重要，思考力比记忆力更重要。
  12. 目无全牛 - 找到其内在的逻辑
  13. 申论考的是什么？ 透过现象看本质，思维能力，情商。
  14. 做一个什么样的学生？
    • 奴：被动的存在
    • 徒：主动，但什么也不知道
    • 工：主动，懂一点皮毛
    • 匠：精细，精致。别人让做，照图纸，没有自己的创新
    • 师：把握规律
    • 家：无人能及
    • 圣：人生 ，全部打通
  15. 申论的道：述材料。 出题者的思路 => 材料 => 答案
  16. 申论考试命题和答题的逻辑：命题专家的抽象 => 演绎成材料的具体 => 考生答案的抽象
  17. 申论答题的问题1: 碎片化的答案。
  18. 申论答题的问题2: 银丝提取不能破坏红木。
  19. 申论辩证法的三个关键词：多分法，结合，具体问题具体分析
    • 多分法
    • 创新是高分的必要条件：原始性创新很难；集成性的创新（结合 => 申论万能思维体系不同角度的结合），结合点：
      • 物质 —- 精神（文化，思想）
      • 市场 —- 政府
      • 管理 —- 服务
      • 法治 —- 人治
      • 开发（当前） —- 保护（长远）
      • 传统 —- 现代
      • 民族 —- 世界
      • 个人 —- 集体
    • 举例：SWOT分析（strength-内因，好，weakness-内因，坏，opportunity-外因，好，threat-外因，坏）
    • 如何创新？ 结合多维度，具体问题具体分析
    • 举例：老漂族 心灵失根导致的精神孤独，后顾有忧导致的精神焦虑。原因：老龄化和城镇化的叠加；现代生活方式与中国传统文化的交融；人口流动与户籍壁垒的交错。
    • 举例：选择性执法好不好？ 1.被动的选择性执法：人财物有限；2. 主动的选择性执法：善意的和恶意的：权力寻租。
  20. 申论的万能思维体系
    • 实务维度：
      • 主体
        • 构成
          • 政府：公共部门的第一部门，其他还有事业单位和非营利组织。政府的价值追求是：公平公正。
          • 企业：市场领域。价值追求：效率。
          • 群众：社会。价值追求：权利。
        • 方面（发挥作用）
          • 利益：即需要
            • 经济利益：钱，物质
            • 政治利益：权利，权力
            • 文化利益：文化
              • 核心层：价值观，精神，道德（2012联考：动物伦理；2014国考：社会心态，幸福，慢生活；2014年4月联考：文化输出和文化拿来）
              • 载体：物质和非物质（2013国考：文化遗产的保护）
              • 制度：体制，机制。
                • 风俗习惯
                • 法律，制度
                  • 文化事业：公共文化服务
                  • 文化产业：市场性产品
                  • 文化交流：2012.9 联考
            • 社会利益：民生
            • 生态利益
          • 思想
            • 理念
            • 意识，责任，大局，环保
            • 常识，知识
          • 行为素质
            • 常识，知识
            • 道德，身体，业务素质
            • 心理素质
      • 制度
        • 规则：法律法规，静态，单一，规定
        • 机制：系统，动态的制度，实用化
        • 体制：雷区，政治体制不要碰。（宪政，普世价值，价值事实，价值观）管理体制的三大问题
          1. 多头管理。如食品安全中，工商，药监。。。
          2. 权责不对等。城管很冤枉：相对集中的行政处罚权， 近400项职责
          3. 职责不清。新一届政府提出”权力清单“。
      • 技术
        • 硬技术
          • 技术手段（物联网）
          • 人：人才，队伍，干部，专家，编制
          • 财：资金，经费，支出，投入，预算。预算分三种：
            • 财政投入
            • 银行贷款，发债
            • 利用民间资本，社会资本（BT／BOT项目）
          • 物：设施，设备，机构，组织
        • 软技术：管理
          • 规划，决策
          • 施行，执行
          • 监督，检查，评估（标准）
          • 总结，反思
      • 环境
        • 人文环境，社会氛围
        • 自然环境
    • 时间维度
      • 过去，现在，未来
      • 事前，事中，事后
    • 空间维度
      • 物理空间
        • 民族
        • 世界
      • 思维空间
        • 内因，外因。如三农问题：输血式：贷款；造血式：农民主动性
        • 主观，客观
    • 价值维度
      • 利：经验，意义，成绩，积极
      • 弊：教训，危害，问题，消极
  21. 写申论：唯物论 —-> 述材料（辩证法） —-> 升华（申论想象力）
  22. 申论想象力：和而不同，嬉笑但不怒骂，有个性但不张扬
  23. 申论写作误区1：申论是写作考试：申论其实是公共管理的对策写作与论证，为什么？怎么做
  24. 申论写作误区2:申论是杂文评论（偏激） 申论与杂文的区别：1.语言简单朴素而不失形象生动；2.角度：公共管理
  25. 申论写作误区3:申论是学术论文
  26. 申论写作误区4:申论是政府文件（大而空 => 要落地）
  27. 举例：玉林狗肉节 讨论的是人和动物的关系问题。动物伦理的主流观点：1.绝不能虐待或者虐杀动物；2.吃不吃狗肉，不能用道德绑架；3.吃不吃狗肉，与个人的情感体验相关，也尊重爱狗人士；4.民俗与传统要尊重，但不宜推广。
  28. 申论考试的本质是 公务员实际管理工作的全真模拟，公共管理侧重于实战应用。
    • 申：引申，说清楚：特定事实，该事实来自于问题的设问。
      • 特定事实的表现：是什么？有什么影响（意义和危害）？
      • 特定事实的原因
      • 特定事实的对策
    • 论：论证，说服别人
      • 表现的准确性
      • 原因的针对性
      • 对策的有效性
  29. 申论高手
    • 官员：求真务实，管理政府
    • 学者：深入研究，深刻观点
    • 评论员：一针见血，高度概括（白岩松，杨禹，许树泽）
  30. 基本题型
    • 阅读理解能力
    • 综合分析能力
    • 提出和解决问题能力
    • 文字表达能力
  31. 申论阅卷
    • 阅卷主体：命题者是高校年轻老师，博士生，政治学相关专业，师大容易出题
    • 阅卷地点：电脑阅卷，扫描进去就没有格子
    • 阅卷流程：点阅，试评：细化评卷答案；命题思路；得分标准：采分制（权威答案和参考答案）
    • 阅卷老师一天要看1000篇左右，一般习惯：
      • 看卷面，字数（不勾划）
      • 再看内容：看标题，是否找到题眼（20分）
      • 再看结构，每段一句话 （25分）
      • 再看语言生动 （30分）
      • 再看论据是否支持论点
  32. 写作题的秘密：形式大于内容 卷面+语言 ； 文章内容：表达形式+逻辑形式
  33. 心态准备：太重要了！
  34. 任何职业都是利弊参半的！ 公务员优势：稳定和社会福利等；公务员劣势：劳神费力（勾心斗角）；极其不自由；高危行业；本领恐惧。 当公务员就别想发财，否则就是对自己的不负责。 特别注意：别把理想的设计当成现实。
  35. 职业：从政，从学，从商。
  36. 顺其自然：努力之后，接受一切结果。 使劲往上爬；爬到哪算哪。
  37. 舍得；门槛（跨过去是门，跨不过去是槛）；能耐（有能力，耐得住寂寞）；命运（命是失败者的借口，运是成功者的谦词）
  38. 是非审之于心，毁誉听之于人，得失安之于数。
  39. 中国文化中最精华的部分就是：修心。
  40. 享福：享同样的福，受同样的罪。

《钟君讲申论》务实部分：做题的思路和方法

阅读理解的基本套路

  1. 关键：用材料说话，述材料
  2. 两种问题意识
    • 具体的问题意识：阅读快；找点准；审题（非常重要！！）
    • 抽象的问题意识：材料主题；主题的表现；原因；对策。
  3. 审题的目的：破题
    找出潜台词，还原题型。申论考的就是情商。情商包括：
    • 自知之明
    • 自控力
    • 同理心
    • 适度的热情
  4. 限定材料 —- 答案来源； 作答任务 —- 题干涉及的特定事实； 回答体例 —- 表达形式。
  5. 阅读材料的“三遍四步法” ： 一般要用30-40分钟读三遍材料
    • 第一遍 采点勾画 —- 精读：找到采分点，关键字，句
    • 第二遍 归纳大意 —- 泛读
      • 核心任务是：挑选出最重要的关键词
    • 第三遍 梳理整合 —- 泛读
    • 第一步 勾画关键词或关键字，采点
      • 首尾句原则
      • 关联词：语法中的强调作用
        • 因为 所以 ：因果
        • 虽然 但是 其实 于是 ：转折-表示强调
        • 同时，还，此外：表示并列
      • 观点性原则
        • 词汇包括：指出，强调，希望，期望，据悉，认为，断定，说等
        • 调查结论
        • 政府官员观点：这种都是正确
        • 专家观点：辩证看
        • 民众观点：利益的诉求表达
        • 重大公共管理的决策，都有一个：“两害相权取其轻”
      • 常见词原则（高频词）
        • 根源，危害，教育，体制，领导，法律法规，监督，落实，经验，教训（详见申论万能思维体系）
    • 第二步：归纳大意
      • 核心任务是：挑选出最重要的关键词
    • 第三步：分门别类
      • 横向原则（单一性概括）：并列关系（政治，经济，文化，社会，生态）；不同角度的并列（万能思维体系）
      • 纵向原则（综合性概括）：表现，原因，对策
      • 多层次原则：这个很少考
    • 举例：工作的实体维度包括：主体，技术，制度，环境。
    • 整合方法：
      • 同类合并：相同概念的词合并在一起
      • 异类罗列：中心，室，协会
      • 概括主旨：
    • 第四步：概括主要内容的两个模式
      • 倡议书，发言，阅读材料，调查报告

分析题的基本套路

  • 特别的说明：所有的分析题首先是概括题，概括占比50%以上。这就是申论唯物观。述材料。
  • 阐释题，对比题，评价题，启示类题

分析题型1: 阐释类题目

  • 回答 “是什么” “为什么”
  • 技巧：直接概括+间接引申
  • 和概括题的差别：
    • 概括题：复杂内容简单化
    • 阐释题：简单内容复杂化

分析题型2: 设喻类题型

  • 要将比喻的概念转化成具有实质性意义的概念
    • 一是把表面的直接含义弄清楚（美国人民，非洲难民，鲸鱼的比喻意义）
    • 二是对字面含义背后的实质内容进行揭示（经济发展与生态环境的关系）
    • 三是要明确隐含意义，表明态度。G所长想要说而未说出来的意识（在怒江水电开发上的态度）
    • 四是要在此基础上说开去。如何协调生态与经济的关系问题。
  • 一定要把喻体说清楚

分析题型3: 原因分析题（非常重要！！！）

  • 技巧：概括 + 辨证思维
    • 为什么污染如此严重？ 因-》果
    • 为什么要治理污染？政论文的思路：为什么要干什么？回答：污染严重性和紧迫性。治理污染的可行性。治理污染的重要性，意义或好处。
    • 为什么要考公务员？ 面临的挑战和困难。可行性。重要性：意义和好处。
  • 抽象后的题型1: 过往性原因分析。 为什么现在会这样？重点分析过去发生了什么。
    1. 分析方法：万能思维体系：实物思维：主体，技术，制度，环境。
    2. 其他几种角度的思维：主客观分析，主观：利益，思想，素质。客观：制度，技术，环境。
    3. 内外因分析（思维空间角度）：内因（治本），外因（治标）
  • 抽象后的题型2: 未来性原因分析。 为什么将来要那样？也是必要性分析。
    1. 严重性和迫切性： 问题和挑战
    2. 可行性： 有利条件，成绩
    3. 重要性：意义，好处。

分析题型4: 比较分析题

  • 比较分析题 = 2-3 道概括题
  • 技巧：深入理解材料内容，准确把握其区别和联系，关键是找到比较点。
  • 第一步：概括不同观点，模式，要素。
  • 第二步：才是比较。
  • 题型1:关系分析题（2010年北京题：发展经济，保护环境，保障健康的内在关系）
  • 题型2:焦点题。技巧：焦点或争议的表达方式（eg 活熊取胆）

分析题型5: 评价题（非常重要！！！）

  • 技巧：辩证思维，多角度看问题（面试中结构化面试最难的都是综合分析题：难在思维养成和热点的敏感性）
  • 基层公务员向上渠道：遴选，选调：分析题和对策性的议论文写作
  • 辩证法：多分法+结合+具体问题具体分析

  • 题型1: 分析评论两个或两个以上看似矛盾的观点

    • 思维模式1: 优劣势模式 特别强调两者的结合！（2005年国考 解决农村农民问题的两种思路）

    • 思维模式2: 相互依存的模式 （eg：经济发展和环境保护）

    • 申论考试中常用的几对矛盾范畴

      • 物质-精神（文化，思想）：物质基础+思想保证，智力支持，精神动力
      • 市场-政府： 看不见的手+看得见的手 宏观调控，监管，公共服务，保护公平+资源配置中起决定性作用（市场领域的负面清单：法无禁止即可为；政府领域的正面清单：法无规定不可为） 公共产品的定价方面：2010年联考 水资源定价
      • 管理-服务：政府的两条腿，政府主导对社会管理，社会主导对政府服务。 管理提供的是一种秩序；服务那是政府的职责所在，政府存在的价值。 管理是为了更好的服务，服务需要管理的保障，加强管理中完善服务，要在完善服务中加强管理。 服务指的是：公共服务，即解决老百姓的民生问题。九大方面：就业，医疗，社保，教育，住房，环境，公共安全等。 管理—》治理：自上而下+共同的。都有一个管字在里面，那么怎么管？管治：法治：制度化管理；稳定的，长效的，可靠的。和人治：行政化管理；随意的，暂时的，短期的，选择性的，一阵风，运动式，集中治理整顿。
      • 法治-人治（德） 地铁禁食是否需要立法？ 那地铁上脱鞋，搂抱。。。是否需要写进法律？ 法律能不能做得到？ 法律：他律，刚性，强制。德治：自律，全方位的约束。佛教：特别强调德治和法治的结合，戒+定=慧。 怎么解决地铁禁食：曝光台；
      • 开发（当前）-保护（长远）：周生贤观点。
      • 传统-现代：不尊重历史，就没有现代，历史是现代的借鉴
      • 民族-世界
      • 个人-集体
      • 公平-效率（政府和市场）：初次分配兼顾公平和效率，再分配要更加重视公平。

总结

解决问题（提出对策）的基本套路

决策程序

  1. 诊断问题所在，确定决策目标；
  2. 收集尽可能完备的资料与信息，为制定政策提供充分的信息保障；
  3. 依据尽可能完备与可靠的信息，对发展的趋势变化做出准确的预测；
  4. 拟定各种备选方案；
  5. 对各种备选方案进行可行性和不可行性评价；
  6. 从各种备选方案中选出最优方案。

首先，要根据材料找到特定社会现象表现出来的消极影响。这部分工作实际上在阅读材料和概括主要内容时已经完成。
其次，根据对这种消极负面影响的分析，找到其产生的原因。
过往性原因分析方法 申论万能思维体系

  • 申论作文的味道：复合味
    • 公文的中正：理性的态度，中立的立场，全面思考，辩证的思维。 中庸是一种境界，做人：外圆内方，内外兼修。 要有一个大我：家国天下的情怀。
    • 杂文的深刻：杂文的大家能够抓住事物的本质，但不要偏激的语言。
    • 散文的生动：形象的将问题阐述出来。第一语言：第二语言：作家才会有，作家才能把这种美表达出来，这种语言需要训练—靠的是积累。文学生动的思维方式。诗歌中最重要的是异象，给你相同的感受，比如“舌尖上的安全”。
    • 论文的严谨：强调的是一种推理，严谨和准确。
  • 申论的形象说法
    • 带着镣铐的舞蹈
    • 申论的想象力：和（镣铐）而不同（舞蹈）
  • 写策论 or 写政论？
    • 都是议论文，没有这种区分
  • 申论的分类：根据问题确定论证的方式
    • 一般议论文：提出问题，分析问题，解决问题。怎么了，为什么，怎么办？
      • 策论文：重点是写怎么办
      • 政论文：重点是写为什么
      • 评论文：重点是写分析中的怎么看
    • 公文
      • 法定公文
      • 非法定公文

  • 优秀申论文章的五度空间

    • 高度：考虑问题有大局意识。不要简单的喊口号，要自然的流露出来。eg 商品高度包装。
    • 深度：看问题要看本质。
      • 春节不放假：
        • 社会账： 熟人社会和陌生人社会。传统的走动
        • 文化账： 文化的三个层次，核心层，载体层，制度层。文化的载体和制度。传承的是中国的传统文化价值观。
    • 广度：强调的是一种视野（空间：地理空间和思维空间；时间的视野），而不是视力。eg 两只饿狼看见草地的表现。
      • 把思维打开的方法：万能思维体系的结合。比如食品安全：空间：中国，日本，时间：过去，现在，未来。这样就有6种。再把价值维度加进去，过去，现在和未来中国，日本在食品安全的经验教训，挑战和机遇。制度，环境。
      • 务工荒：春节过后，年轻人不愿意去东南沿海打工，为什么？分析其原因
        • 人口荒：劳动力总量还没有减少，还没有达到刘易斯拐点（社科院：二十年后）。低小散追求高大壮，低端小型分散的企业追求的是高端大规模青壮年的劳动力。非常严重的劳动力资源浪费。现在日本老龄化国家好多都是老年人在工作。1988年到达人口顶峰，一直在下降。29岁。年轻的劳动力越来越少。青壮年劳动力荒。
        • 权利荒：进程务工人员在城市的权利不能保障
        • 机会荒：沿海地区的机会越来越少，家乡创业机会越来越多
        • 价值荒：主体意识的觉醒，想法越来越多。
    • 效度／力度：解决问题的具体办法
      • eg 提高出租车收益：时间：拥堵时间提高出租车费用，空间维度：拥堵区域和非拥堵区域。
    • 风度：让你的文章有文采，生动

  • 写作的审题：一类卷：30分以上 二类卷：20-30 三类卷：10-20分 四类卷：10分以下 审题正确，达到20分左右

    • 找题眼： 论点就是在破题中形成的。两个论点的原则：1.唯物论原则 2.系统整体性原则
    • 定角色：站在什么角度上写文章，既是小我，也是大我，就是大局观的问题。
    • 知体例：议论文
    • 明字数：字数差太多，一下就打到了四类卷，允许一点误差，多或少一两行。
      • eg 2014国考地市级 幸福充满着缺陷
        • 社会心理：器质性的不健康，神经病；价值观：亚健康。写的是价值观。幸福是什么？幸福和缺陷的关系。幸福和心理健康的关系。
      • eg 2014国考副省 生活要慢下来：题眼是慢下来。慢的是什么？读材料就知道慢的是什么
      • eg 2014年4月联考题目 输出价值观 。
        • 国家价值观 是什么
        • 输出价值观
          • 为什么
          • 怎么输出
      • eg 拿来与输出 题眼：是“与”。潜台词就是关系。评论式议论文：一对矛盾的关系，优劣势的模式A好坏+B好坏=》A+B 互补性模式：没有A，B怎么样，没有B，A怎么样。必须强调两者之间的内在关系。
      • eg 中国人的文化自尊： 尊重：内在 自我：外界冲击
  • 构思—-谋篇布局
    • 确定立意和论点
    • 确定论证逻辑和结构
      • 策论式
      • 政论式
      • 评论式
    • 论点—-立意
      • 整体性原则
        • 把申论材料看作统一的整体，要有问题意识，对于材料要有整体性把握。
      • 唯物论原则
        • 要特别善于从材料中获取立意的灵感。 特别注意！！！抄材料原文
      • 确定论点的公式
        • 时代背景+鲜明观点+一般套话+特定事实
      • 突出论点的方法
        • 前后照应
        • 1 文章的题目就是论点：突出出来
        • 2 第二段单独成段，点明论点。第一段概述材料的主要内容，紧接着第二段要点明论点。
        • 3 文章的每个分论点要扣住总论点，每段开头或结尾回扣总论点。
        • 4 最后一段总结总论点。

  • 论证的本质和逻辑

    • 说服别人 ==> 为什么我是对的
    • 论证的本质就是原因分析：
      • 过往性原因分析：策论式文章。 为什么会这样？—- 万能思维体系
      • 未来性原因分析：政论式文章。 为什么要那样？—- 未来性原因分析

    • 论证的三种逻辑和结构

      • 策论式论证逻辑和结构

        • 提出问题，怎么了？
        • 分析问题，为什么？

        • 解决问题，如何做？（重点） 1000字文章大约500字，字数上多点。

        • 第一段 开头说明怎么了，提出观点

        • 第二段 说明为什么会这样，分析原因：原因1；原因2；原因3
        • 第三段 对策A-对应的是原因1。发散性思维+五个要素（主体，对象，内容，方式，目的） 更饱满些的话，多写一写目的：为什么要这样？
        • 第四段 对策B-对应的是原因2
        • 第五段 对策C-对应的是原因3

        • 第六段 结尾，适当小结或展望未来

        • 一 展开式论证

          • 发散性思维+五个要素（主体，对象，内容，方式，目的） 更饱满些的话，多写一写目的：为什么要这样？

        • 二 必要性论证

          • 严重性，紧迫性，可行性，重要性

        • 策论式的范文：如何破解打车难？—- 要有实招

          • 背景（央视新华网热点，总书记视察公司）+论点：政策，技术，市场，社会四管齐下。
          • 分析原因？不考虑管理等因素，主要有以下三点：
            • 结构性不足：出租车的绝对数量不足；
            • 摩擦性不足：空驶率。人找不到车，车找不到人
            • 出租车利益不足。只保第三方，下雨下雪天出租车出车，风险大。
          • 过渡句：过渡到解决问题。
          • 依靠政策和市场两只手，解决第三个原因。解决份子钱：工资集体协商制度，公司公开运营成本，建立出租车特别的保险。
          • 解决信息的不对称：打车软件。
          • 拼车等

      • 政论式的论证逻辑

        • 提出问题，怎么了？
        • 分析问题，为什么？（重点）

        • 解决问题，如何做？

        • eg 为什么要大力发展农村文化？

        • 未来性原因分析：严重性，紧迫性；有利条件；重要性，好处和意义。

        • 政论式论证结构：落脚也要落脚到做法

          • 第一段 开头说明怎么了，要怎样，提出观点。 eg 大力发展农村文化
          • 第二段 出现问题的严重性和迫切性
          • 第三段 解决问题的有利条件
          • 第四段 解决问题意义A： eg 提高农民的素质
          • 第五段 解决问题意义B： eg 推广农业科技，帮助农民增收致富
          • 第六段 解决问题意义C： eg 抵制赌博，净化农村的社会氛围，促进农村的社会稳定
          • 第七段 结尾，总的对策，适当展望。 eg 提高农村文化的做法

        • 在基本逻辑清晰的情况下，基本原则：文无定法，自圆其说。可以侧重于不同的部分。

        • 意义的论证方式：方法之一种
          • 起 eg 买乐高
          • 承 eg 买乐高怎样
          • 转 eg 不买的话，我就。。。
          • 合 eg 你还是给我买一个乐高吧，好处多多
        • 政论式范文：警惕社会之霾 ：要这个就要写意义，不要这个就要写危害。 钟君：《社会之霾：中国社会风险研究》
          • 重点谈危害。训练目无全牛的感觉。

      • 评论式论证逻辑

        • 提出问题，怎么了
        • 分析问题，怎么看？（重点）
          • 一对矛盾：A好坏+B好坏=A+B；A->B,B->A,A+B; eg 环境保护和经济发展
          • 一个现象：多角度+本质

        • 解决问题，如何做？

        • 评论式论证结构

          • 第一段 开头说明怎么了，提出观点
          • 第二段 看问题的角度A
          • 第三段 看问题的角度B
          • 第四段 看问题的角度C
          • 第五段 看问题的角度D
          • 第六段 解决问题的对策
          • 第七段 结尾，适当展望。
        • 评论式的范文：德与得
          • 表达方式：同声异形
          • 强调的是要道德的得到
        • 评论式的范文：城镇人口首超农村
          • 坏的方面：严峻挑战
          • 好的方面：提供动力
      • 论据（平时积累过程中需要积累的，习近平的讲话中：至理名言。芝麻大的窟窿能透进斗大的风。谚语用一两句即可。公信力：塔西陀效益，不管政府做什么，都是在做坏事。习近平：破窗效应：坏的行为示范作用，一个车的车窗坏了一块，其他的车窗很快被砸坏；鲢鱼效益；蝴蝶效应。）
        • 一个非常重要的论证方法：论据论证法。例证。
        • 摆事实：事实材料：具体的事例／概括的事实／统计数字／亲身经历和感受
        • 讲道理：理论材料：经典著作／至理名言／民间的谚语和俗语／科学上的公理，规律等等
        • 每段中都要有一个论据，材料之外的论据。

    • 申论文章的五个得分突破点

      • 一 提纲或框架
        • 一定要在草稿纸上写出来。阅卷习惯：标题，首段，首句。。
        • 提纲的内容（文章的框架）：把你思考的过程书面化
          • 标题
          • 开头／总论点
          • 分论点及其对应的材料／每一段的开头和结尾
          • 论证结构
          • 结尾

      • 二 标题

        • 最好是中心论点，或者反映中心论点。主题力求醒目，简练。
        • 批判式标题：不要。。。
        • 点睛式标题：
        • 对比式标题：在。。。之间 “把中国制作变为优秀制造”
        • 引申式标题
        • 回避总论点的万能标题：由。。。现象引起的思考
        • 追问式的标题
        • 副标题不要（加了副标题也不加分，错了要扣最多3分）
        • 居中显示
        • 10个字以内

      • 三 开头

        • 单刀直入，简明明快，开门见山，直接揭示文章主题。
          • 事件+影响+观点
          • 排比式的方式
          • 对比式的
          • 引文式
          • 点题式

      • 四 结尾

        • 事件+意义／或者回扣主题
        • 言简意赅，情真意切，给人回味
        • 综上所述，总而言之，从以上的分析我们知道

      • 五 流畅的过渡句

      • 审题-破题-立意 总论点贯穿始终

      • 构思（论证结构，相应的论据，表达）
      • 装修-五个得分突破点

人生的辩证法

申论的几个关键点：

  • 逻辑关系非常重要
  • 综合性概括题：找到特定事实 ===> 表现，原因，对策
  • 答案的形成即 ===> 落地：
    • 总括句（特定事实）
    • 分述细节信息（表现，原因，对策或并列的各方面）
    • 概括题答题模式图解
  • 概括主要内容的基本标准：
    • 全面：主要信息 ===> 采分点要全 信息点：事实性和价值性
    • 准确：
    • 简明：材料中的关键词
    • 流畅：有层次，有条理，有逻辑：综合性（表现，原因，对策）；单一性（并列的各种关系）
  • 抄原材料的句子是大忌
  • 问题意识-审题-破题：
    • 采点：首尾，关联，观点性，常见词
    • 大意
    • 分类：申论的逻辑：表现，原因，对策；申论的思维：单一性概括：申论万能思维
    • 成文
  • 看文章：就要看内在结构。 目无全牛
