幼儿园入园前的准备及入园后的膳食和保健

幼儿园卫生保健基本内容

  1. 健康检查：入园体检，返园体检，晨检及全日观察
  2. 传染病防控：发现传染病及时进行隔离，直至传染病痊愈。

儿童保健

8月份打疫苗针后，将儿童保健手册领回来，幼儿园要做每半年的体检。

打针的绿本需要交到幼儿园

  1. 每年大小体检
  2. 特殊儿童管理（肥胖儿童，体弱儿童，贫血儿童管理，五官保健）
  3. 体质测定
  4. 预防接种通知（打针的绿本需要交到幼儿园）
  5. 配合牙防组涂氟(幼儿园会有知情同意书)

科学膳食

  1. 核心：均衡，科学，合理
  2. 三餐两点：加餐：早上：牛奶，酸奶，自制点心；下午：水果
  3. 食谱都是统一的，从制作到发放都是严格的带量标准。
  4. 膳食制定和营养计算：针对每个孩子的摄入量是均衡的
  5. 进餐：幼儿园小班进餐：饭菜分开，小勺用餐，再喝汤，进餐时间20-30分钟。不能泡饭，饭菜混进去，主要是咀嚼不细。
  6. 培养良好的进餐习惯
  7. 固定开饭时间，可以喝点酸奶和牛奶
  8. 保护孩子的食欲：孩子在幼儿园吃的过程需要一个适应过程，（胆小，挑事等因素），在家千万不要吃太多，影响其食欲。
  9. 保护好的习惯：安静的吃饭，注意力集中在饭菜上，好的进餐习惯，否则容易导致厌食。
  10. 做饭时，要提供种类丰富的食物，比如喜欢吃土豆等，长期不吃导致偏食，要多做一些种类和口味。

体质锻炼

  1. 上下午各一小时，每天2小时，保健医要跟踪监督的，穿着是否合理，小背心和短裤：穿短袖和七分裤，容易磕破，睡觉时才穿小背心。不建议穿系扣子的衬衫。
  2. 小班下学期和中班下学期：体质测试，40分，32分优秀，30-32分，20分-30分及格。包括身高，体重， 坐位体前屈，立定跳远，网球掷远，双脚连续跳，10米折返跑，走平衡木。

亲子园和幼儿园保健工作的衔接

  1. 传染病的预防
  2. 培养好习惯：卫生习惯，饮食习惯，作息习惯，儿童保健
  3. 特殊儿童：龋齿，视力异常，肥胖
  4. 9点睡觉，7点起床
  5. 六步洗手法：肥皂，15-20秒

幼儿入园前应做哪些准备

五年期间，课程不断调研丰富，北郊，中关村和杏林湾三个校区的小班老师调研。

幼儿园阶段：各种行为习惯的形成，认知不是最重要的

一日作息时间

07:50-08:00 入园高峰，8：15之前吃早餐完毕 小厨房：交往能力的训练

养成早睡早起的习惯

⚠️的要从家里现在开始培养

  • 07:30 教师上班
  • 07:50-08:20 早餐（吃早餐应在8点前入园）⚠️
  • 8:20-9:10 区域活动，加餐喝奶，喝水
  • 9:10-10:00 教学活动，喝水
  • 10:00-11:00 户外活动（做操，组织游戏）
  • 11:00-11:20 餐前游戏（老师带领大家，手指游戏，听故事，老师消毒桌面）
  • 11:20-12:00 午餐，餐后休息，准备午睡（清水漱口，如厕等）⚠️
  • 12:00-14:00 午睡（3个小时以上）⚠️
  • 14:00-14:30 午点，喝水
  • 14:30-15:30 教学活动，区域活动，喝水
  • 15:30-16:30 户外活动
  • 16:30-16:50 餐前游戏
  • 16:50-17:20 晚餐
  • 17:20-18:00 静园

孩子入园前应做哪些准备

开始锻炼孩子自己穿衣服，等

九点左右睡觉，早起：七点多起床，以后早上六点多起床。中午睡不着没关系，但12：00-14:00一定要安静，在床上听故事，建议听故事，讲故事会有一问一答会更兴奋。不能睡到自然醒，到点叫醒。午睡是最危险的时候：老师经常巡视，尿床，拿着异物等。

一定要独立睡觉，适应能力和好的作息时间会让孩子睡觉。

小东西一定要拿掉，比如小皮筋绕到手指上截肢。路上不要捡东西，小石头，小铁丝，一定不要拿。

孩子心理准备：入园焦虑期长度与能力强弱相关

好的习惯都是来自于家庭教养

上学后直接影响学习，孩子初期培养习惯非常重要

  • 自理能力的训练：重视培养孩子的各项生活自理能力，增强孩子的自信心，减少入园后的焦虑和自卑（穿脱衣裤鞋子，喝水，洗手等）
  • 睡眠训练：在了解幼儿园作息制度后，有意识的进行作息调整（养成早睡早起，每天午睡，独立入睡）；改掉一些睡觉时的习惯（睡觉，拿东西，穿纸尿裤）
  • 吃饭训练：固定的时间和地点让孩子吃饭，不喂饭，养成进餐的好习惯（安静进餐，饭菜搭配吃）；少盛一点饭，吃完再添，让孩子吃完后享受一种成就感，进餐时间不宜过慢（30分钟内）
  • 如厕：练习使用蹲坑如厕，练习腿部力量
  • 大声清楚表达自己的意愿（比如尿裤子，肚子疼等，口渴了）
  • 和同伴交往：互相友好表达自己的意愿
  • 心理方面的准备
    • 正面语言引导孩子，幼儿园是快乐的地方，不能拿幼儿园和老师吓唬孩子
    • 入园前，有条件可带孩子熟悉环境，减少陌生感，遇到幼儿园改建，寒暑假期间进行改建。
    • 培养孩子的独立能力，在没有成人注视的情况，减轻孩子的依赖感（安静游戏，看书等）

家长应该做好哪些准备

  • 家长调整心态，给予孩子积极暗示

    轻松点，正面的东西，比如我们去幼儿园吧。要有一定的耐心和“狠心”（比如不想去幼儿园，哭了就不去了，这样试探家长等，这时候要有狠心），建立起老师和幼儿的依恋关系，克服焦虑情绪（忌：对老师没有足够的信任，入园时不离不舍，中途探望，轻易许诺等）

    送孩子不要在门里看，这个门其实是孩子的希望之门。送进去就离开，有问题老师会打电话。

  • 为孩子准备合适的衣物

    准备易于孩子穿脱的四季衣裤和鞋子（姓名标识，轻便衣裤，鞋一定要合脚防滑，卫生间瓷砖。鞋一定要穿合适的⚠️），便于教师及时给孩子更换。衣服上最好绣上名字。

  • 和老师多沟通交流，如实反映孩子的情况

    如实反映情况：过敏史，疾病史，惊厥，习惯性脱臼等。行为习惯一定要及时沟通，为了改正其习惯。（比如麦苗儿打人咬人等习惯）包括性格上面：积极和老师配合。

  • 遇事理性处理各种问题

    孩子之间的矛盾：孩子非常单纯，都要宽容。经过一次打闹后，还成为朋友。别因为孩子的矛盾，家长之间产生矛盾。别断了孩子结交朋友的路子。
    孩子磕着碰着，相互理解。

亲子园七八月课程安排

家长在孩子上课时不要上课！⚠️

  1. 自理能力的培养：穿脱衣服，鞋子。进餐拿勺。洗手等
  2. 学习友好的交流和合作：分享玩具，如何在群体中表达自己的意愿
  3. 规则意识的培养：学习和同伴拉手，站队，行进走，学会安静等待（不是都是第一名），物归原处等。
  4. 好习惯的养成。正确的挥笔姿势，坐姿及各种行为等。(直接影响用眼卫生)
  5. 让孩子适应不同的老师
  6. 引导孩子专注听讲，独立完成自己的事情。 （家长要后退，孩子独立完成。）
  7. 其他事宜：8.21-25毕业典礼。

有一天我会放手，因为路要自己走！
